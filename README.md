---

## Setup Instructions

### 1. Clone / create the repository

Create a repository on GitHub named exactly:
```
payelsantra.github.io
```
Upload all files from this folder to that repository.

### 2. Add your photo

Place your photo as:
```
images/profile.jpg
```
Recommended: square crop, at least 400×400px.

### 3. Add your CV

Place your CV PDF as:
```
files/Resume_PAYEL_SANTRA.pdf
```

### 4. Enable GitHub Pages

- Go to your repository → **Settings** → **Pages**
- Under *Source*, select **Deploy from a branch**
- Branch: `main` / `master`, folder: `/ (root)`
- Click **Save**

Your site will be live at `https://payelsantra.github.io` within a few minutes.

## Customization

| What to change | Where |
|---|---|
| Profile photo | `images/profile.jpg` |
| CV / Resume | `files/Resume_PAYEL_SANTRA.pdf` |
| Bio text | `index.html` → `#about` section |
| Publications | `index.html` → `#publications` section |
| News items | `index.html` → `#news` section |
| Accent color | `css/style.css` → `--accent` variable |
| Google Scholar link | `index.html` → update `href` in social links |

## Tech Stack

- Pure HTML + CSS + Vanilla JS — no frameworks, no build step
- Google Fonts: DM Serif Display + DM Sans + JetBrains Mono
- Fully responsive (mobile-friendly)
- Works offline except for Google Fonts
